## üöÄ Installation

### From Shopify Theme Store
1. Go to your Shopify Admin
2. Navigate to Online Store > Themes
3. Click "Add theme" > "Explore free themes" or "Upload theme"
4. Search for "MegaStore Pro"
5. Click "Add theme" and publish

### Manual Installation
1. Download the theme ZIP file
2. In Shopify Admin, go to Online Store > Themes
3. Click "Upload theme"
4. Select the ZIP file and upload
5. Customize and publish
